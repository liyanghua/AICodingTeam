# Benchmark: Opportunity Insight Skill Creator

## Purpose

This benchmark evaluates the **Skill Creator**, not the final opportunity-card answer.

It tests whether the Skill Creator can compile ecommerce opportunity insight knowledge into a complete Skill Pack with:

- Scenario boundary
- Data contract
- Knowledge contract
- Tool contract
- Execution plan
- Output schema
- Evaluation contract
- Failure policy
- RSI feedback loop
- App generation readiness

## Pass Standard

- All hard gates must pass.
- Total rubric score must be >= 80.
- Score >= 90 is excellent.
