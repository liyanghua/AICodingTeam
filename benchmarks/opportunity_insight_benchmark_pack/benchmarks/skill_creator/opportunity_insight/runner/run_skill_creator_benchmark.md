# How to Run This Benchmark

## Runner Steps

1. Read `benchmark.yaml`.
2. Load all files under `input/`.
3. Send benchmark input to Skill Creator.
4. Generate `generated_skills/ecommerce/opportunity_insight/`.
5. Check required file tree.
6. Check hard gates.
7. Run capability checks.
8. Score by rubric.
9. Produce `benchmark_report.json` and `benchmark_report.md`.

## Important Rule

Do not require exact match with `reference_skill_pack/`. The reference pack is a guide, not the only acceptable answer.
