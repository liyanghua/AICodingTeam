# Opportunity Insight Skill Creator Benchmark Pack

This package defines a benchmark for testing whether a Skill Creator can generate a high-quality ecommerce `opportunity_insight` Skill Pack.

Core benchmark question:

> Given ecommerce business scenario, data dictionary, knowledge seed, claim pack seed, tool manifest and sample data, can the Skill Creator generate a reusable, executable, evaluable and evolvable Business Skill Pack?

Directory entry:

```text
benchmarks/skill_creator/opportunity_insight/
```
