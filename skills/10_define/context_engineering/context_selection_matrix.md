# Context Selection Matrix

| Need | Include | Exclude |
| --- | --- | --- |
| Planning | PRD, acceptance criteria, related specs | Raw logs, full source dumps |
| Coding | Slice, relevant source, tests, allowed paths | Unrelated modules |
| Debugging | Failing command, focused logs, recent diff | Successful historical logs |
| Review | Diff, acceptance criteria, tests run | Chat history |
