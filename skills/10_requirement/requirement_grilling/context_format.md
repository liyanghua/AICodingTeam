# Context Format

## Goal

- 一句话业务结果。
- 本次最小可交付版本。

## Users

- Primary user
- Secondary user
- Reviewer / approver

## Scope

- In scope
- Out of scope
- Deferred

## Terms

- Shared glossary entries

## Workflow

- User entry point
- Main steps
- Completion signal

## Data Objects

- Object name
- Required fields
- Status values
- Sensitive fields

## UI States

- Loading
- Empty
- Error
- Success
- Needs human confirmation

## Acceptance Signals

- Automated tests
- Review checklist
- Manual validation

## Open Questions

- Unknowns that still need decisions
