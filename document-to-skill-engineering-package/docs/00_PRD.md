# PRD：Document-to-Skill Compiler

## 1. 背景

业务团队存在大量策略文档，例如市场洞察、竞品分析、价格带布局、主图策划、产品升级、达人建联等。这些文档包含业务经验、执行流程、数据来源、判断标准和输出模板，但目前只能被人阅读，不能被 Agent Runtime 稳定执行。

## 2. 产品目标

把任意业务策略文档转换为可执行 Skill 包，使 Agent 能够：

1. 理解业务目标。
2. 拆解业务流程。
3. 明确每一步的数据需求。
4. 自动匹配工具。
5. 生成结构化输出。
6. 绑定证据链。
7. 被评测、回放和持续优化。

## 3. MVP 范围

以《市场分析洞察元策略》为第一个样例，实现：

- 文档解析。
- Strategy IR 生成。
- Market Insight Skill 包生成。
- Data Requirement 抽取。
- Tool Binding 模板生成。
- Output Schema 生成。
- Eval Rule 生成。

## 4. 非目标

MVP 不直接完成：

- 所有真实外部平台数据采集。
- 生意参谋等平台账号登录自动化。
- 完整企业级权限系统。
- 完整可视化工作台。

## 5. 成功标准

- 给定市场洞察策略文档，能够生成完整 Skill 包。
- 生成内容中至少包含 10 个流程节点、10 类输出物、核心数据需求、基础评测规则。
- `pytest` 通过。
- 下游 AI-coding 可以基于生成包继续实现真实工具。
